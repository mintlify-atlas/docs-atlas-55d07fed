`@aws-amplify/ui-vue` provides Amplify UI components for Vue 3 applications. The primary component is `Authenticator`, which delivers a complete sign-in, sign-up, and MFA experience connected to Amazon Cognito. Components can be imported individually or registered globally via the Vue plugin.

**Current version:** `4.4.2`  
**Requires:** Vue `^3.0`

## Basic usage

Import the `Authenticator` component directly in your SFC:

```vue App.vue
<script setup lang="ts">
import { Authenticator } from '@aws-amplify/ui-vue';
import '@aws-amplify/ui-vue/styles.css';
</script>

<template>
  <Authenticator>
    <template v-slot="{ user, signOut }">
      <h1>Hello, {{ user?.username }}</h1>
      <button @click="signOut">Sign out</button>
    </template>
  </Authenticator>
</template>
```

## Using the composable

`useAuthenticator` gives you reactive access to the current authentication state from anywhere in your component tree:

```vue components/UserMenu.vue
<script setup lang="ts">
import { useAuthenticator } from '@aws-amplify/ui-vue';

const { user, signOut } = useAuthenticator();
</script>

<template>
  <div v-if="user">
    <span>{{ user.username }}</span>
    <button @click="signOut">Sign out</button>
  </div>
</template>
```

<Note>
  `useAuthenticator` must be used inside a component that is a descendant of `<Authenticator>` or `<Authenticator.Provider>`.
</Note>

## Available exports

| Export | Kind | Description |
|---|---|---|
| `Authenticator` | Component | The full Authenticator UI |
| `useAuthenticator` | Composable | Reactive auth state and controls |
| `SignIn` | Component | Sign-in step sub-component |
| `SignUp` | Component | Sign-up step sub-component |
| `ForgotPassword` | Component | Password reset step |
| `ConfirmSignIn` | Component | MFA confirmation step |
| `ConfirmSignUp` | Component | Sign-up verification step |
| `ConfirmResetPassword` | Component | Reset password confirmation |
| `ForceNewPassword` | Component | Force-new-password step |
| `SetupEmail` | Component | Email MFA setup |
| `FederatedSignIn` | Component | Social/federated sign-in buttons |
| `AuthenticatorSignUpFormFields` | Component | Customizable sign-up fields |
| `AuthenticatorForceNewPasswordFormFields` | Component | Force-new-password fields |
| `translations` | Object | I18n strings for overriding labels |
