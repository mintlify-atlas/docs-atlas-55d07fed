`@aws-amplify/ui-react` is the primary Amplify UI package. It provides a full set of primitive components, connected components (such as `Authenticator` and `AccountSettings`), theming utilities, and hooks — all built for React 16, 17, 18, and 19.

**Current version:** `6.15.2`

## Basic setup

Import the stylesheet and configure Amplify before rendering your app:

```tsx main.tsx
import { Amplify } from 'aws-amplify';
import '@aws-amplify/ui-react/styles.css';
import outputs from '../amplify_outputs.json';

Amplify.configure(outputs);
```

Then use any component directly in your JSX:

```tsx App.tsx
import { Authenticator } from '@aws-amplify/ui-react';

export default function App() {
  return (
    <Authenticator>
      {({ signOut, user }) => (
        <main>
          <h1>Hello, {user?.username}</h1>
          <button onClick={signOut}>Sign out</button>
        </main>
      )}
    </Authenticator>
  );
}
```

## Hooks

```tsx
import { useTheme, useBreakpointValue } from '@aws-amplify/ui-react';

function MyComponent() {
  const { tokens } = useTheme();
  const variant = useBreakpointValue({ base: 'small', large: 'large' });

  return <div style={{ color: tokens.colors.brand.primary[80].value }}>{variant}</div>;
}
```

## Theming

```tsx
import { createTheme, ThemeProvider } from '@aws-amplify/ui-react';

const theme = createTheme({
  name: 'my-theme',
  tokens: {
    colors: {
      brand: {
        primary: {
          80: { value: '#007bff' },
        },
      },
    },
  },
});

export default function App() {
  return (
    <ThemeProvider theme={theme}>
      {/* your app */}
    </ThemeProvider>
  );
}
```
