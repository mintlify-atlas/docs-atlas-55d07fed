`@aws-amplify/ui-angular` brings Amplify UI to Angular applications. The package provides the `Authenticator` component — a fully managed sign-in, sign-up, and MFA flow connected to Amazon Cognito — along with supporting primitives and the `AmplifyAuthenticatorModule` for NgModule-based apps.

**Current version:** `5.3.2`  
**Requires:** Angular `>= 19.0.0`

## Basic usage

After completing [Angular setup](/frameworks/angular/setup), use the `amplify-authenticator` element in your templates:

```html app.component.html
<amplify-authenticator>
  <ng-template amplifySlot="authenticated" let-user="user" let-signOut="signOut">
    <h1>Welcome, {{ user.username }}!</h1>
    <button (click)="signOut()">Sign out</button>
  </ng-template>
</amplify-authenticator>
```

## Customizing form fields

Use the `amplifySlot` directive to override sign-up or force-new-password form fields:

```html app.component.html
<amplify-authenticator>
  <ng-template amplifySlot="sign-up-fields">
    <amplify-sign-up-form-fields></amplify-sign-up-form-fields>
    <!-- add custom fields here -->
  </ng-template>
</amplify-authenticator>
```

## Accessing auth state

Inject `AuthenticatorService` to read the current auth state or trigger sign-out programmatically:

```typescript app.component.ts
import { Component } from '@angular/core';
import { AuthenticatorService } from '@aws-amplify/ui-angular';

@Component({ selector: 'app-root', templateUrl: './app.component.html' })
export class AppComponent {
  constructor(public authenticator: AuthenticatorService) {}

  get user() {
    return this.authenticator.user;
  }

  signOut() {
    this.authenticator.signOut();
  }
}
```

## Exported API

| Export | Kind | Description |
|---|---|---|
| `AmplifyAuthenticatorModule` | NgModule | Declares all Authenticator components and primitives |
| `AuthenticatorComponent` | Component | The `<amplify-authenticator>` element |
| `AuthenticatorService` | Service | Reactive access to auth state |
| `AmplifySlotDirective` | Directive | `amplifySlot` attribute for template customization |
| `SignUpFormFieldsComponent` | Component | Customizable sign-up field set |
| `ForceNewPasswordFormFieldsComponent` | Component | Force-new-password field set |
| `TextFieldComponent` | Component | Text input primitive |
| `CheckboxComponent` | Component | Checkbox primitive |
| `translations` | Object | I18n translation strings |
