Amplify UI is an open-source component library that makes it easy to build cloud-connected web and mobile applications. It provides both high-level connected components (like a fully-featured Authenticator) and low-level primitives (like Button, TextField, and Flex) — all accessible, customizable, and built to integrate with AWS Amplify.

## What's included

Amplify UI has three categories of components:

**Connected components** handle complex cloud workflows out of the box. Drop in the `Authenticator` and you get a fully-functional sign-in/sign-up flow with MFA, SSO, and passkeys — no backend code required.

**Primitive components** are the building blocks of your UI: layout containers (`Flex`, `Grid`, `View`), form elements (`TextField`, `SelectField`, `CheckboxField`), and display elements (`Badge`, `Alert`, `Table`). They follow WCAG accessibility guidelines and integrate with the Amplify theming system.

**The theme system** lets you customize every design token — colors, spacing, typography, radii — globally or per-component, with full dark mode support.

## Getting started

<Steps>
  <Step title="Install the package">
    Install `@aws-amplify/ui-react` (or your framework's package) and `aws-amplify`.
  </Step>
  <Step title="Configure Amplify">
    Call `Amplify.configure()` with your backend outputs at the root of your app.
  </Step>
  <Step title="Add a component">
    Import and render a connected component like `<Authenticator>` or a primitive like `<Button>`.
  </Step>
  <Step title="Customize the theme">
    Use `createTheme` and `<ThemeProvider>` to match your brand.
  </Step>
</Steps>
