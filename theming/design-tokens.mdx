Design tokens are the named values that define the visual properties of Amplify UI. Every spacing value, color, font size, and border radius is a token. Tokens compile to CSS custom properties, so you can override them in a theme or consume them directly in your own CSS.

## Token naming convention

A token's CSS variable name is derived from its path in the theme object, prefixed with `--amplify-` and joined with hyphens:

```
tokens.colors.font.primary  →  --amplify-colors-font-primary
tokens.space.medium         →  --amplify-space-medium
tokens.fontSizes.large      →  --amplify-font-sizes-large
tokens.radii.medium         →  --amplify-radii-medium
```

## Overriding tokens in `createTheme`

Pass a partial `tokens` object to `createTheme`. Only the keys you provide are overridden; everything else falls back to the default theme.

```tsx
import { createTheme } from '@aws-amplify/ui';

const theme = createTheme({
  name: 'my-theme',
  tokens: {
    colors: {
      font: {
        primary: { value: '#1a1a2e' },
      },
      background: {
        primary: { value: '#f8f9ff' },
      },
    },
    space: {
      medium: { value: '1.25rem' },
    },
    radii: {
      medium: { value: '0.375rem' },
    },
  },
});
```

## Using tokens in custom CSS

Reference the compiled CSS variables in your own stylesheets:

```css
.my-card {
  background-color: var(--amplify-colors-background-primary);
  border: var(--amplify-border-widths-small) solid var(--amplify-colors-border-primary);
  border-radius: var(--amplify-radii-medium);
  padding: var(--amplify-space-medium);
  box-shadow: var(--amplify-shadows-medium);
  color: var(--amplify-colors-font-primary);
  font-size: var(--amplify-font-sizes-medium);
}
```

## Token references

Tokens can reference other tokens using the `{path}` syntax. The compiler resolves these references at build time and generates a CSS variable reference (`var(--amplify-...)`) in the output:

```typescript
const theme = createTheme({
  name: 'my-theme',
  tokens: {
    colors: {
      font: {
        // Reference another token by path
        primary: { value: '{colors.neutral.100.value}' },
      },
      background: {
        primary: { value: '{colors.white.value}' },
      },
    },
  },
});
```

This compiles to:

```css
[data-amplify-theme="my-theme"] {
  --amplify-colors-font-primary: var(--amplify-colors-neutral-100);
  --amplify-colors-background-primary: var(--amplify-colors-white);
}
```
