<Note>
  Amplify UI requires `aws-amplify` v6 or later. If you're using an older version, [upgrade to v6](https://docs.amplify.aws/react/start/manual-installation/) before continuing.
</Note>

## Choose your framework

Amplify UI ships a dedicated package for each supported framework. Pick the one that matches your project.

<Steps>

<Step title="Install packages">

Install the Amplify UI package for your framework alongside `aws-amplify`.

<CodeGroup>

```bash React (npm)
npm install @aws-amplify/ui-react aws-amplify
```

```bash React (yarn)
yarn add @aws-amplify/ui-react aws-amplify
```

```bash React (pnpm)
pnpm add @aws-amplify/ui-react aws-amplify
```

```bash Angular (npm)
npm install @aws-amplify/ui-angular aws-amplify
```

```bash Angular (yarn)
yarn add @aws-amplify/ui-angular aws-amplify
```

```bash Angular (pnpm)
pnpm add @aws-amplify/ui-angular aws-amplify
```

```bash Vue (npm)
npm install @aws-amplify/ui-vue aws-amplify
```

```bash Vue (yarn)
yarn add @aws-amplify/ui-vue aws-amplify
```

```bash Vue (pnpm)
pnpm add @aws-amplify/ui-vue aws-amplify
```

```bash React Native (npm)
npm install @aws-amplify/ui-react-native aws-amplify
```

```bash React Native (yarn)
yarn add @aws-amplify/ui-react-native aws-amplify
```

```bash React Native (pnpm)
pnpm add @aws-amplify/ui-react-native aws-amplify
```

</CodeGroup>

</Step>

<Step title="Import the stylesheet">

Amplify UI components require the default stylesheet to render correctly. Import it once at the root of your application.

<Tabs>
  <Tab title="React">
    Add the import to your app entry point (e.g. `src/main.tsx` or `src/index.tsx`):

    ```tsx src/main.tsx
    import '@aws-amplify/ui-react/styles.css';
    ```

    <Tip>
      Amplify UI also ships a CSS Layers variant at `@aws-amplify/ui-react/styles.layer.css`. Use this if you need to control cascade ordering with your own CSS.
    </Tip>
  </Tab>
  <Tab title="Angular">
    Add the stylesheet to `angular.json` under `projects > architect > build > options > styles`:

    ```json angular.json
    "styles": [
      "node_modules/@aws-amplify/ui-angular/theme.css"
    ]
    ```

    Or import it globally in `src/styles.css`:

    ```css src/styles.css
    @import '@aws-amplify/ui-angular/theme.css';
    ```
  </Tab>
  <Tab title="Vue">
    Add the import to your app entry point (e.g. `src/main.ts`):

    ```ts src/main.ts
    import '@aws-amplify/ui-vue/styles.css';
    ```
  </Tab>
  <Tab title="React Native">
    React Native does not use CSS. Amplify UI for React Native applies styles via the React Native StyleSheet API — no additional import is needed.
  </Tab>
</Tabs>

</Step>

<Step title="Wrap your app with ThemeProvider (optional)">

For React, you can optionally wrap your application with `ThemeProvider` to apply a custom theme or control the color mode.

```tsx src/main.tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import { ThemeProvider } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <ThemeProvider colorMode="system">
      <App />
    </ThemeProvider>
  </React.StrictMode>
);
```

</Step>

</Steps>

## Package versions

| Framework    | Package                       | Version  |
| ------------ | ----------------------------- | -------- |
| React        | `@aws-amplify/ui-react`       | 6.15.2   |
| Angular      | `@aws-amplify/ui-angular`     | 5.3.2    |
| Vue          | `@aws-amplify/ui-vue`         | 4.4.2    |
| React Native | `@aws-amplify/ui-react-native`| —        |
| Core         | `@aws-amplify/ui`             | 6.15.2   |

<Note>
  `@aws-amplify/ui` is the shared core package. It is installed automatically as a dependency of the framework-specific packages — you do not need to install it directly.
</Note>
